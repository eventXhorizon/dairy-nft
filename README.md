# Quickstart

### 1. Install Dependencies
```bash
npm install
```

### 2. Deploy the Contract
Build the contract and deploy it in a testnet account
```bash
npm run deploy
```

### 3. Start the Frontend
Start the web application to interact with your smart contract 
```bash
npm run start
```
